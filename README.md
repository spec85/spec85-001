# 拟物天气卡片 - 四季变幻

一个使用纯 HTML、CSS 和 JavaScript 构建的拟物化天气卡片展示项目，融合了四季元素与丰富的交互体验。

## ✨ 项目特色

### 🎨 拟物化设计
- 精美的渐变背景和多层阴影效果
- 半透明毛玻璃卡片质感
- 每个季节独特的配色方案
- 细腻的凹凸光影效果

### 🌸 四季元素展示
| 季节 | 代表植物 | 天气状态 | 特色元素 |
|------|----------|----------|----------|
| 🌱 春 | 樱花 | 晴朗微风 | 粉色花瓣飘落 |
| ☀️ 夏 | 荷花 | 雷阵雨 | 雨丝 + 闪电特效 |
| 🍂 秋 | 枫叶 | 多云转晴 | 金黄落叶旋转 |
| ❄️ 冬 | 梅花 | 小雪纷飞 | 洁白雪花飘落 |

### 🎯 交互动画
- **卡片点击**：触发脉冲动画 + 季节粒子效果
- **天气图标点击**：图标弹跳动画 + 强化粒子效果
- **悬停效果**：卡片上浮、图标旋转、植物摇摆
- **环境动画**：太阳旋转、云朵漂浮、闪电闪烁、雪花飘动
- **背景粒子**：定时触发的微妙季节氛围动画

### 🔊 Web Audio API 音效
使用纯代码合成的实时音频，无需外部音频文件：

- **春季 - 风铃声**：清脆的正弦波和弦，五音递进
- **夏季 - 雷雨声**：白噪声模拟雨声 + 低频隆隆雷声
- **秋季 - 踩树叶声**：沙沙声 + 高频脆响
- **冬季 - 踩雪声**：柔和白噪声 + 低频挤压声

## 🚀 使用方法

1. **直接打开**：双击 `index.html` 在浏览器中打开
2. **本地服务**（推荐）：
   ```bash
   # 使用 Python
   python -m http.server 8000
   
   # 或使用 Node.js
   npx serve
   ```
3. 访问 `http://localhost:8000`

## 🛠️ 技术实现

### HTML 结构
- 语义化的卡片布局
- 内联 SVG 图标（天气图标 + 植物元素）
- 响应式网格布局

### CSS 特性
- CSS Grid 响应式布局
- `@keyframes` 动画：旋转、漂浮、闪烁、下落
- `box-shadow` 多层拟物阴影
- `backdrop-filter` 毛玻璃效果
- `clip-path` 自定义形状（枫叶）
- CSS 变量和渐变

### JavaScript 功能
- 面向对象的 `WeatherCardApp` 类
- 事件委托和事件冒泡管理
- 动态 DOM 元素创建与清理
- **Web Audio API**：
  - `OscillatorNode` 合成音调
  - `GainNode` 音量包络控制
  - `BiquadFilterNode` 频率滤波
  - `AudioBuffer` 白噪声生成
  - 正弦波、三角波、方波合成

## 📝 自定义扩展

### 修改季节数据
在 `index.html` 中找到对应卡片，修改：
- 日期：`<span class="date">`
- 温度：`<span class="temp-value">`
- 天气描述：`<div class="weather-desc">`
- 湿度/风速：`.humidity .value` / `.wind .value`

### 调整动画参数
在 `style.css` 中修改 `@keyframes` 的时长或缓动函数：
```css
@keyframes petal-fall {
    animation-duration: 3s;  /* 修改下落时长 */
}
```

### 调整音效参数
在 `script.js` 中修改对应方法：
- 频率数组：调整音调高低
- gain 值：调整音量大小
- 时长参数：调整音效持续时间

## 🎵 音效合成原理

### 风铃声（春季）
- 五声音阶频率：C5, E5, G5, C6, E6
- 依次触发，间隔 0.15 秒
- 指数衰减包络，模拟余音缭绕

### 雷雨声（夏季）
- 白噪声 + 高通滤波 = 雨声
- 白噪声 + 低通滤波 + 频率下滑 = 雷声
- 两次雷声间隔 1.5 秒

### 踩树叶声（秋季）
- 白噪声 + 带通滤波 = 沙沙声
- 方波振荡器 + 快速频率下滑 = 脆响
- 5 次脆响间隔 0.4 秒

### 踩雪声（冬季）
- 白噪声 + 低通滤波 = 轻柔背景声
- 三角波振荡器 + 频率下滑 = 挤压声
- 4 次挤压声间隔 0.8 秒
